# Website Analytics Reporting Project

## Overview
This is a self-initiated portfolio project created to practice website analytics, reporting, and data interpretation skills.

## Objective
The goal of this project is to analyze website performance data, identify unusual trends or anomalies, and suggest opportunities for A/B testing.

## Business Questions
- Which pages or channels perform best?
- Where do users drop off in the funnel?
- Are there unusual changes in traffic or conversions?
- What areas could be tested to improve conversion rate?

## Tools Used
- Google Sheets / Excel
- Data cleaning
- KPI reporting
- Basic analytics interpretation

## Deliverables
- KPI summary
- Data quality or anomaly observations
- Funnel and performance insights
- A/B testing recommendations
